
# select日期选择器组件

### Attributes

| 参数        |    说明    |   类型 |                                                                   可选值 | 默认值 |
| ----------- | :--------: | -----: | -----------------------------------------------------------------------: | -----: |
| v-model     |   绑定值   |    any |                                                                       -- |     -- |
| type        | 选择器类型 | string | year/month/date/dates/ week/datetime/datetimerange/ daterange/monthrange |     -- |
| placeholder |  提示信息  | string |                                                                       -- |     -- |
