# 图片上传

- 功能: 上传单个图片 再次上传会修改已上传的图片
- 说明: 目前图片是死的 后期配合后端以达到效果

### Attributes

| 参数        |          说明          |   类型 | 可选值 |       默认值 |
| ----------- | :--------------------: | -----: | -----: | -----------: |
| v-model     |         绑定值         |    any |     -- |           -- |
| content     |          内容          |    any |     -- |           -- |
| width   | 容器宽度(px) | string |     -- |           178 |
| height   | 容器高度(px) | string |     -- |           178 |

>基本上包含了 `elementUI` 属性 ,`api`参考地址: https://element.eleme.cn/#/zh-CN/component/upload
